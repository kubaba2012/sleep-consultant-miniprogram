# 助眠师微信小程序界面原型

一个专注于声音助眠和正念冥想的微信小程序界面原型。

## 🎯 项目简介

助眠师微信小程序致力于通过专业的助眠音频和正念冥想引导，帮助用户改善睡眠质量，快速入眠。

## 🌟 核心功能

### 1. **助眠音频库**
- 自然声音（雨声、海浪、森林、白噪音等）
- 正念冥想（呼吸冥想、身体扫描、渐进式肌肉放松等）
- 助眠音乐（古典音乐、轻音乐、脑波音乐等）

### 2. **个性化推荐**
- 基于用户偏好的音频推荐
- 睡眠质量追踪和分析
- 智能播放列表生成

### 3. **冥想练习**
- 正念冥想课程
- 呼吸训练指导
- 睡眠准备仪式

### 4. **睡眠辅助工具**
- 睡眠定时器（自动关闭音频）
- 白噪音混合器（自定义声音组合）
- 睡眠记录和统计

### 5. **用户社区**
- 睡眠经验分享
- 冥想打卡挑战
- 专业睡眠建议

## 🎨 设计风格

**视觉理念**：
- 主色调：蓝紫色系（营造宁静氛围）
- 辅助色：绿色和白色（传达健康与舒适）
- 字体：无衬线字体（清晰易读）
- 布局：简洁优雅（大量留白）

**交互设计**：
- 平滑过渡动画
- 轻量级反馈
- 响应式设计
- 卡片式布局

## 🚀 技术实现

### 前端技术栈
- HTML5 + CSS3 + JavaScript
- 响应式设计
- 移动优先开发
- 无框架轻量级实现

### 外部资源
- 使用 picsum.photos 提供的随机图片作为音频封面
- 使用 Google Fonts 提供的 Noto Sans SC 字体

## 📱 界面预览

### 首页
- 快速功能入口
- 今日推荐轮播
- 热门分类导航
- 专家睡眠建议

### 播放页
- 沉浸式音频播放
- 播放控制功能
- 音量调节
- 定时播放设置

### 分类页
- 音频分类筛选
- 搜索功能
- 分类内容浏览

### 个人中心
- 用户信息管理
- 收藏内容
- 睡眠记录
- 设置功能

## 🔧 开发和部署

### 本地开发
```bash
# 启动本地服务器
python3 -m http.server 8000

# 或使用 HTTPS 服务器
python3 https_server.py
```

### Vercel 部署
项目已配置 Vercel CI/CD，可以直接部署到 Vercel 平台。
